Welcome to Isaac Checklist

This is a personal progress tracker for The Binding of Isaac: Repentance completion marks. Each row is a completion mark and shows which unlock you still need for every character.

Getting started

  • Switch between Regular and Tainted characters at the top.
  • Your overall progress is shown in the header.
  • Progress is saved automatically in your browser.
@if (layout.isMobile()) {

On mobile

  • Use the arrows or tap the character name to switch characters.
  • Tap an achievement row to mark or unmark its unlock.
} @else {

On desktop

  • Browse the wiki-style table with every character visible at once.
  • Click any unlock cell to toggle it on or off.
}
